# Negativos Essenciais — Serviços Locais de Emergência

## Por Que Negativos São Críticos

Sem negativos, você paga por cliques de:
- Pessoas querendo emprego ("vaga desentupidora")
- Concorrentes pesquisando seu anúncio
- Usuários que querem fazer sozinhos ("como desentupir")
- Queries fora da área de atuação (outra cidade)
- Buscas educacionais/informacionais sem intenção de contratar

**Regra do Pedro Sobral:** revise os search terms a cada 7 dias nos primeiros 3 meses. Depois, a cada 14 dias.

## Lista Mestra de Negativos — Todos os Serviços

```
emprego, vaga, currículo, trabalhar, contratar funcionário
curso, treinamento, formação, aprender, tutorial, como fazer
youtube, video, aula
grátis, gratuito, de graça, sem custo
preço tabela, tabela de preços (se não quiser mostrar preço)
reclamação, reclamar, procon, reclamações
wiki, wikipedia
```

## Como Adicionar Negativos Corretamente

### Níveis de Aplicação
1. **Lista de negativos compartilhada** (aplicar em todas as campanhas): termos genéricos indesejados
2. **Campanha** (aplicar só em campanhas específicas): termos fora do escopo daquela campanha
3. **Grupo de anúncios** (raramente): controle granular

### Match Types de Negativos
- **[Exact Negative]** para termos que podem aparecer em contextos bons: `[curso]` negativiza só "curso" isolado, mas não "curso de canos" em um contexto diferente
- **Phrase Negative** para combinações: `"como fazer"`, `"faça você mesmo"`
- **Broad Negative** raramente — pode negativizar termos úteis

### Protocolo Semanal de Revisão
1. Abre relatório Search Terms
2. Ordena por Custo (maior para menor)
3. Analisa os top 50 termos
4. Qualquer termo com ≥ 2 cliques e sem conversão que seja claramente irrelevante → negativar
5. Termos com conversão → considerar adicionar como keyword exata se ainda não tiver

## Negativos Geográficos

Para serviços locais: negativar cidades vizinhas onde não atende.
```
[Nome das cidades fora da área de atendimento]
```

**Cuidado:** não negativar termos de localidade genérica se você atende a cidade toda ("são paulo", "sp").
