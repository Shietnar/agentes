# Headlines Testadas — Fórmulas que Convertem para Emergência

## Fórmulas de Headline por Tipo de Urgência

### Fórmula 1: Serviço + Localidade + Tempo
```
[Serviço] em [Cidade] — [Prazo]
"Desentupidora em São Paulo — Atende em 1 Hora"
"Chaveiro em Campinas — 30 Minutos"
"Gasista no Rio — Hoje Mesmo"
```

### Fórmula 2: Problema + Solução Imediata
```
[Problema do usuário]? [Solução imediata]
"Cano Entupido? Resolvemos Hoje"
"Trancado? Abrimos em 30 Minutos"
"Vazamento de Gás? Técnico Agora"
```

### Fórmula 3: Número + Benefício + Prazo
```
[Número específico] + [Benefício] + [Urgência]
"45 Minutos — Chegamos no Seu Endereço"
"24h — Atendemos Qualquer Emergência"
"15 Anos — Resolução Garantida"
```

### Fórmula 4: Garantia Explícita
```
[Ação] + [Garantia] + [CTA]
"Orçamento Grátis — Preço Fixo — Ligue Já"
"Problema Resolvido ou Não Cobra"
"Garantia de 30 Dias — Chame Agora"
```

### Fórmula 5: Social Proof + Especialidade
```
[Social Proof] + [Especialidade]
"+5.000 Atendimentos em [Cidade]"
"4.9★ Google — Desentupidora [Cidade]"
"Empresa Local Há 12 Anos"
```

## Headlines que Não Funcionam (e Por Quê)

### Evitar: Superlativos sem prova
❌ "A Melhor Desentupidora de SP" → não crível sem prova
❌ "Serviço de Qualidade" → vago, todo mundo diz isso
❌ "Profissionalismo e Dedicação" → abstrato, não resolve o problema urgente

### Evitar: Foco no serviço, não no cliente
❌ "Nossa Empresa Tem 20 Anos de Experiência" → "nossa" foca na empresa
✅ "20 Anos Resolvendo Emergências em [Cidade]" → foca no benefício

### Evitar: Urgência sem especificidade
❌ "Atendimento Rápido" → o que é rápido? 1h? 3h?
✅ "Chegamos em Até 1 Hora" → específico e verificável

## Headlines por Ângulo Emocional

### Ângulo: Medo de Pagar Caro
```
"Orçamento Grátis — Sem Surpresas no Final"
"Preço Justo — Sem Letra Miúda"
"Cobro Pelo Serviço, Não Pela Visita"
```

### Ângulo: Medo de Ser Enganado
```
"Empresa Registrada — CNPJ [número]"
"Técnicos Identificados e Certificados"
"4.9★ em 200+ Avaliações Verificadas"
```

### Ângulo: Urgência de Resolução
```
"Resolve Hoje — Sem Agendamento"
"Chamo Agora — Atendo Às 23h"
"Domingo e Feriado — Sem Adicional"
```

### Ângulo: Conveniência
```
"PIX, Cartão ou Dinheiro — Sem Complicação"
"Nota Fiscal Garantida"
"Sem Taxa de Visita — Pago Só Se Resolver"
```

## Descrições para RSA — Estrutura Ideal

**Regra:** cada descrição deve funcionar SOZINHA, sem depender das outras.

### Descrição modelo 1 (benefício + prova):
"Desentupimento de esgoto, pia e caixa de gordura. Chegamos em até 1 hora. Orçamento gratuito e preço fechado antes de começar. Avaliação 4.9★ no Google."

### Descrição modelo 2 (urgência + garantia):
"Atendemos às 2h da manhã, sábados, domingos e feriados. Técnicos certificados, equipamento profissional. Garantia de 30 dias no serviço executado."

### Descrição modelo 3 (local + confiança):
"Empresa local em [Cidade] há [X] anos. Conhecemos cada bairro. Mais de [N] famílias atendidas. CNPJ [número]. Aceitamos qualquer forma de pagamento."

## Sitelinks com Alta Conversão

```
Ligue Agora (link para tel:)
WhatsApp Direto (link para wa.me)
Nossos Serviços
Área de Atendimento
Avaliações dos Clientes
Orçamento Online
```

**Sitelinks que não convertem para emergência:**
- "Quem Somos" — o cliente não quer saber sua história, quer saber se você resolve o problema
- "Blog" — irrelevante em momento de emergência
- "Parcerias" — idem
